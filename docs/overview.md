# Project Overview
Status: Draft (to be updated during implementation)

## What this project builds
- Centralized Windows telemetry pipeline using WEF + Sysmon
- Downstream ingestion into Wazuh, Elastic, and Splunk Free for triage

## Scope
- Endpoints: AD-WIN10 / AD-WIN11
- Collector: (AD-DC01 or dedicated WEC)
- Outputs: Wazuh, Elastic, Splunk Free
